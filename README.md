# Enheduanna — An Intertextual and Multimodal Writing Platform

A prototype of a collaborative digital environment where Mário Cesariny's poem *"You are welcome to Elsinore"* serves as a generative matrix for an intertextual mosaic. Inspired by the first named author in history, this platform weaves fragments into an infinite *alicatado* of multimodal blocks, mediated by AI agents.

Built for the Digital Writing Workshop (Master's in Creative Writing, University of Coimbra).

## Features

- Multimodal blocks (text, image, audio, video, PDF)
- Three mediation agents: Enheduanna (weaver), Wanderer (walker), Ogum (inventor)
- Local saving and selective sharing
- Integration with external tools (Udio, DALL-E, Twine, Poemário)

## Current Status

Functional prototype with local storage (localStorage). In development: cloud integration for data sharing.
